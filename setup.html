🚀 Instructions de déploiement

1

Téléchargez le fichier config.js

Cliquez "Télécharger config.js" ci-dessus. Ce fichier contient toute la configuration de votre CRM.

2

Déposez les fichiers sur votre serveur

Placez dans le même dossier : config.js · commande.html · crm.html · prospect.html

3

Exécutez le SQL dans Supabase

Si ce n'est pas encore fait, allez dans Supabase → SQL Editor et exécutez le SQL de l'étape 2.

4

Ouvrez crm.html

Votre CRM est opérationnel. Le bon de commande et le CRM sont synchronisés avec Supabase.
